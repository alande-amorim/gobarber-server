# Aula 01 - configurando estrutura

```bash
$ yarn add express
$ mkdir ./src
$ touch ./src/app.js
$ touch ./src/server.js
$ touch ./src/routes.js
```
### ./src/app.js:
Arquivo de configuração da aplicação

### ./src/server.js:
Inicializa o servidor

### ./src/routes.js:
Define as rotas da aplicação

## Executando a aplicação:
```bash
$ node ./src/server.js
```
